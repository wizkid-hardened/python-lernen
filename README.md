# Python Lernprojekt

Mein persönliches Lernprojekt zur Vorbereitung auf eine Ausbildung im IT-Bereich (Fachinformatiker für Anwendungsentwicklung oder Systemintegration). 

Python dient mir hierbei als Brücke: Zum Verstehen von Software-Logik für die Anwendungsentwicklung und als mächtiges Werkzeug für Automatisierung und Scripting in der Systemintegration.

## 📂 Geplante & gelernte Themen

* **Grundlagen:** Variablen, Schleifen (Loops), Bedingungen und Funktionen
* **Dateiverarbeitung:** Einlesen, Filtern und Schreiben von System- und Logdaten
* **Objektorientierte Programmierung (OOP):** Klassen, Module und saubere Code-Struktur
* **Security & Automation:** Automatisierung von IT-Aufgaben und Security-Scripting

## 🛠️ Tech Stack & Umgebung

* **Sprache:** Python 3.x
* **Betriebssystem:** Debian Linux 🐧 (Terminal-fokussiert)
* **Versionsverwaltung:** Git & GitHub
